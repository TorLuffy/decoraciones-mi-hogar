# Decoraciones Mi Hogar

Web catálogo de **Decoraciones Mi Hogar** — tienda de decoración, ropa de cama,
cortinas, menaje y pequeños electrodomésticos en San Cristóbal de La Laguna (Tenerife).

- Para editar productos y datos de la tienda: archivo `js/datos.js`.
- Guía de uso sencilla: `GUIA-DE-USO.md`.
